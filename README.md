# world-health-data-
## Build from sources 
1. Clone the repo
```bash
git clone https://github.com/Ishtiaque-146/world-health-data-.git
```
2. Install and activate virtual environment
```bash
python -m venv venv
.\venv\Scripts\activate
```
3. Install dependencies
```bash
pip install -r requirements.txt
```
